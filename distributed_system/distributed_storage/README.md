## 第二部分：分布式存储

### 1、大数据对分布式存储的需求

* 存储容量大
* 高吞吐量
* 高数据可靠性
* 服务高可用(99.95%每年4-5小时不可用)
* 高效运维
* 低成本

### 2、分布式存储系统面临的挑战

* 磁盘错误
* Raid卡故障
    - Raid卡用于高可用节点，用于存储Meta数据
